# Autoo-Do

A minimalist Clockodo crate with automation engine

## Purpose

To develop a Clockodo crate that can be used in different projects to automate things - maybe later on with `no-std` to run on SBC.

## State

Totally WIP

## Preparation

Copy the `.env.example` to `.env` and enter all values there as needed.\
You can get your Clockodo API-Key from your Web-Clockodo-UI under your personal settings.

### Run main test program

If not done already, [install rust](https://rustup.rs).

```bash
cargo run .
```
